# Artefacts Tanzil en attente de publication

Ce répertoire est ignoré par Git. Il contient uniquement les copies verbatim
préparées par `../../tools/build-tanzil-text-packs.sh` et l'avis de licence
Tanzil associé.

Avant une release, vérifier les manifests puis les SHA-256. Ne jamais modifier
un texte téléchargé, ni publier une archive sans l'attribution Tanzil.

Le format Tanzil `txt-2` contient 6 236 versets et un avis de licence en fin
de fichier. Vérifier les deux parties avec
`../../tools/verify-quran-text-pack.sh <fichier.txt>`.
